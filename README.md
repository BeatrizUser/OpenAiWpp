# OpenAiWpp

Esta aplicaçao utiliza WppConnect e Openai para criar uma inteligencia artificial que interage via whatsapp.

## Quer testar?

### Copie a aplicaçao
```sh
git clone https://github.com/BeatrizUser/OpenAiWpp.git
```

### Rode o arquivo main.js
```sh
node main.js
```
